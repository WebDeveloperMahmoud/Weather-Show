#Wethear Show
-Weather Show is a web application that provides real-time weather updates and a 3-day forecast for any city worldwide.
-The user can simply enter the city name, and the app instantly displays detailed temperature data and weather conditions.
#Features
-Search for any city worldwide
-Display current temperature and condition
-Show 3-day weather forecast
-Display humidity, wind speed, and weather icons
-Fully responsive design
-Clean and readable UI
